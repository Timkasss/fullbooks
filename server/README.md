## Description

Back end for our full stack project about books

## Installation

First of all you need to remove yarn.lock from server folder, then execute this bash script:

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
